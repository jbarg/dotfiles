Linux dotfiles
======================

Zur Installation der dotfiles:

```sh
git clone https://github.com/jbarg/dotfiles
cd dotfiles
git pull && git submodule init && git submodule update && git submodule status
```

Um alle *git submodules* zu aktualisieren:

```sh
git pull && git submodule init && git submodule update && git submodule status
```
